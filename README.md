# GitHub Pages — лендинг с фото

## Как залить
1. Создай репозиторий на GitHub (например `photo-landing`).
2. Загрузи туда все файлы из этого архива (корень должен содержать `index.html`).
3. В репозитории: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
4. Ссылка появится через пару минут: `https://<username>.github.io/<repo>/`

## Где менять фото
Файл: `assets/img/photo.jpg`

Если репозиторий будет открытым, лучше не добавлять исходники в огромном размере — эта версия уже оптимизирована под веб.
